# KulturGIS fylr plugin

This plugin is used by the [KulturGIS fylr](https://kulturgis-amh.gbv.de) and adds the following functionality:

* New context menu option for linking find places to an ensemble based on their respective geometries

## Installation

The latest version of this plugin can be found [here](https://github.com/gbv/fylr-plugin-kulturgis/releases/latest/download/KulturGIS.zip).

The [NFIS geometry plugin](https://github.com/programmfabrik/fylr-plugin-custom-data-type-nfis-geometry) needs to be installed for this plugin to work.
